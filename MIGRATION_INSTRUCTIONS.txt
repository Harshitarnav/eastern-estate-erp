⚠️  DATABASE MIGRATION REQUIRED ⚠️

Run this command in your terminal:

cd backend
psql -U your_username -d your_database -f create-roles-system.sql

Replace 'your_username' and 'your_database' with your actual PostgreSQL credentials.

This will create:
- roles table (15 default roles)
- permissions table (60+ permissions)  
- role_permissions table (mappings)
- Add role_id to employees table

You should see INSERT confirmations for each role and permission.
