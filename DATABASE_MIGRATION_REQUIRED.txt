⚠️ DATABASE MIGRATION - MANUAL STEP REQUIRED ⚠️

You need to run this command manually in your terminal:

cd backend
psql -U <your_postgres_username> -d <your_database_name> -f create-roles-system.sql

Example:
psql -U postgres -d eastern_estate_erp -f create-roles-system.sql

Or if using environment variables:
psql -U $DB_USERNAME -d $DB_NAME -f create-roles-system.sql

This will:
✅ Create roles, permissions, role_permissions tables
✅ Insert 15 default roles
✅ Insert 60+ permissions
✅ Set up role-permission mappings
✅ Add role_id column to employees table

After running, you should see:
- CREATE TABLE confirmations
- INSERT 0 1 for each role/permission
- ALTER TABLE confirmation

Then the roles API will be fully functional!
